    数据库URL¶

    Mysql¶

    mysql://machine_name:port/dbname

    默认端口：3306

    参考内容¶

    • 关于MySql链接url参数的设置

    Redis¶

    redis://machine_name:port/dbname

    默认端口：6379

    参考内容¶

    • redis url 格式

    MongoDB¶

    mongodb://[username:password@]host1[:port1][,host2[:port2],...[,hostN[:portN]]][/[database][?options]]

    默认端口：27017
